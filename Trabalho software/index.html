Matérias da Faculdade

  • Construção de Software para Web: Foca no desenvolvimento de aplicações web. Você aprende a criar sites e sistemas usando tecnologias como HTML, CSS e JavaScript
  • Design e Desenvolvimento de Banco de Dados I:Ensina como organizar e armazenar dados de forma eficiente. Você aprende modelagem (diagramas), criação de tabelas e uso de SQL para inserir, consultar e manipular dados.
  • Experiência e Interface com o Usuário (UX/UI):Aborda como criar sistemas fáceis e agradáveis de usar. Envolve design de telas, usabilidade, acessibilidade e foco na experiência do usuário ao interagir com o sistema.
  • Lógica para computação:Desenvolve o raciocínio lógico necessário para programar. Inclui algoritmos, estruturas condicionais (if/else), repetição (loops) e resolução de problemas passo a passo.
  • Fundamentos de Tecnologia da Computação:Apresenta os conceitos básicos da área de TI: hardware, software, sistemas operacionais, redes e como os computadores funcionam no geral.
  • Textos Científicos: Aspectos Metodológicos e Linguísticos:Ensina a produzir trabalhos acadêmicos. Você aprende normas (como ABNT), estrutura de artigos, clareza na escrita e organização de ideias de forma científica.
